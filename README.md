# MVVM News App
Using https://newsapi.org to get data and display it in my application . 

I'm using in this project :
* Room Database
* Api
* Coroutines
* Navigation Component
* Gson
* MVVM architecture
* View Model
* Retrofit
* Glide
* WebView
